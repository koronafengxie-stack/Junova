    <p><b>ユノヴァ共和国</b>（ユノヴァきょうわこく、英: Republic of Junova）、通称<b>ユノヴァ</b>は、大陸中央部に位置する共和制国家。首都はヴァルケン。</p>
    <p>強い秩序志向と、独自の「発展段階論」に基づく民族的選別制度を国是とする。参政権を特定の民族（ユノヴァ民族）に限定する「制限的共和制」を敷いており、国家を精密な機械と見なす<b>ユノヴァ主義（Junovism）</b>を統治理念の柱としている。</p>

    <h2 id="summary">概要</h2>
    <p>ユノヴァ共和国は、大陸中央部の広大な領土を統治する。国家優先・秩序重視・民族的発展段階論を中核とする政治思想「ユノヴァ主義」に基づき、効率的かつ合理的な国家運営を目指す。市民は法的カースト制度により厳格に区分されており、ユノヴァ民族が支配階級を形成している。強力な工業力を背景に、軍事的にも大陸内で一定の影響力を保持する。</p>

    <h2 id="geography">地理</h2>
    <p>東西で気候が大きく異なる。南部には広大な海岸線があり、貿易と海運の中心となっている。</p>
    <ul>
        <li><b>西部：</b> 温暖湿潤気候（Cfa）。居住に適し、政治・文化の中心。</li>
        <li><b>東部：</b> ステップ気候（BS）。乾燥した平原で、鉱山や重工業地帯が広がる。夏季は高温だが風が強く、体感温度は比較的穏やかである。</li>
    </ul>

    <h2 id="economy">経済</h2>
    <p>国家主導の重厚長大産業が経済を支える。
    <ul>
        <li><b>重工業・鉱業：</b> 石炭、鉄鉱石などの豊富な資源を背景とした鉄鋼・機械産業。</li>
        <li><b>軍需産業：</b> 国家機械論に基づく軍事力の維持。</li>
        <li><b>畜産業：</b> ステップ地帯を利用した企業的放牧により、牛肉を中心とした食文化が発展。</li>
    </ul>
    </p>

    <h2 id="symbols">国旗・象徴</h2>
    <p>ユノヴァ共和国の国旗と国章は、国家の基本理念である秩序、統一、そして工業力を象徴するようにアルヴェルト・イェルン自身によって設計された。</p>

    <div class="center-image">
        <img src="Junova_flag.png" alt="ユノヴァ共和国の国旗（拡大）" width="480" height="300">
        <div class="image-caption">ユノヴァ共和国 国旗</div>
    </div>
    <p>国旗は、黒・赤・黒の横三色旗の中央に国章を配置したデザインである。黒は「秩序、国家の伝統、揺るぎない土台」を、赤は「国民の情熱、共同体意識、国家機械を動かすエネルギー」を象徴する。</p>

    <div class="center-image">
        <img src="Junova_emblem.png" alt="ユノヴァ共和国の国章（拡大）" width="200" height="200">
        <div class="image-caption">ユノヴァ共和国 国章</div>
    </div>
    <p>国章は、産業と国家機械論を象徴する白い「歯車」を中心としている。その周囲をユノヴァ語の正式国名「REPUBLIC OF JUNOVA」と、平和と繁栄、そして共同体を象徴する「麦穂とオリーブの枝」が囲む。最上部には、国家の指針となる「単一の理想」を象徴する黄金の星が配置されている。</p>


   <h2 id="history">歴史</h2>
    <p>ユノヴァ共和国の歴史は、812年の王国の成立から現在の共和制に至るまで、約900年に及ぶ複雑な変遷を経て形成された。その歩みは中央集権化、軍事拡張、内戦、そして他国による支配からの独立という劇的な過程を含んでいる。</p>

    <h3 id="history-1">王国成立期（812年 - 10世紀）</h3>
    <p>812年、<b>ロバーツ・マーチ</b>が諸侯を統合し<b>ユノヴァ王国</b>を建国した。初期は分権的な封建体制であり、王権は諸侯との合議によって維持されていた。2代目ヴァリアネス1世の強権政策による廃位など政治的不安定な時期もあったが、3代目ルーマ1世の時代に財政再建と長期安定が実現し、国家基盤が整備された。</p>

    <h3 id="history-2">軍事国家化と拡張（9世紀末 - 10世紀）</h3>
    <p><b>エーレン1世（武士王）</b>の治世において、ユノヴァは明確に軍事国家へと転換した。東方への領土拡張が進められ、防衛線として<b>「エーレンの長城」</b>が建設された。しかし、後のロバーツ2世の浪費により財政が崩壊し、王権の権威は一時的に低下した。</p>

    <h3 id="history-3">法治体制の確立：ルーマ3世大王（978年 - 1001年）</h3>
    <div style="float: right; width: 250px; margin-left: 15px; border: 1px solid #a2a9b1; padding: 5px; background: #f8f9fa; text-align: center;">
        <img src="Ruma_III_of_Junova.png" alt="ルーマ3世大王" width="240">
        <div style="font-size: 0.8em; color: #54595d; margin-top: 5px; padding: 5px;"><b>ルーマ3世大王</b><br>「ユノヴァ憲章」を手にし、法による統治を宣言する姿。</div>
    </div>
    <p>ヴァーダニアン朝第3代国王。即位当時の王国は諸侯の力が強く、統治は慣習に依存していた。ルーマ3世は強権を振るうのではなく、<b>「法と合意」</b>を重視する画期的な統治スタイルを確立した。</p>
    
    <h4>ユノヴァ憲章の制定</h4>
    <p>最大の功績は、990年頃に発布された<b>「ユノヴァ憲章」</b>である。これにより、以下の制度が明文化された。</p>
    <ul>
        <li><b>王権の制限：</b> 王といえども法の下にあることを規定。</li>
        <li><b>権利の保護：</b> 貴族の義務と庶民の権利、課税の制限を明記。</li>
        <li><b>紛争解決：</b> 暴力ではなく法廷による解決を制度化。</li>
    </ul>
    <p>この憲章により、ユノヴァは「慣習的支配」から「法的支配」へと移行し、大陸屈指の先進的な法治国家へと脱皮した。庶民の支持を得た彼の治世は、長期的な政治的安定と経済発展をもたらした。</p>
    <div style="clear: both;"></div>

    <h3 id="history-6">領土的最盛期：ハンファロー2世（1216年 - 1230年）</h3>
    <div style="float: left; width: 250px; margin-right: 15px; border: 1px solid #a2a9b1; padding: 5px; background: #f8f9fa; text-align: center;">
        <img src="Hanfaro_II_of_Junova.png" alt="ハンファロー2世" width="240">
        <div style="font-size: 0.8em; color: #54595d; margin-top: 5px; padding: 5px;"><b>ハンファロー2世（盾王）</b><br>背後に秩序の歯車を背負い、王笏を掲げる最盛期の肖像。</div>
    </div>
    <p>ヴァーダニアン朝第14代国王。停滞していた国家を再興し、軍事的才覚によってユノヴァを大陸最大の帝国へと押し上げた。巧みな演説で評議会から戦費を引き出し、前例のない規模の軍事行動を展開した。</p>
    
    <h4>東方遠征と最大版図</h4>
    <p>大規模な東方遠征によって広大なステップ地帯を制圧。これにより王国の領土は現在の共和国の<b>1.5〜1.8倍</b>という史上最大規模に達した。また、新たな防衛線<b>「ハンファローの長城」</b>を築き、東西を結ぶ交易路を確立して軍需産業を活性化させた。</p>
    
    <h4>拡張の代償</h4>
    <p>一方で、急激な拡張は統治コストを増大させ、辺境での反乱や軍部の政治発言力強化という影を落とした。彼の死後、この不安定要素が噴出し、王国は内戦の時代へと突入することになる。</p>
    <div style="clear: both;"></div>

    <h3 id="history-7">軍部台頭と内戦（13世紀中盤 - 後半）</h3>
    <p>ハンファロー2世が遺した強大な軍事力は、平時において政治を圧迫し始めた。13世紀後半、軍部と評議会の対立は決定定的となり、ニードリック3世の暗殺を契機に内戦が勃発。それから国家は長期の内戦状態へと陥り国家の統治機構は崩壊し、経済・インフラは甚大な打撃を受けた。</p>

    <h3 id="history-8">再建期：アウスト朝（13世紀末 - 14世紀）</h3>
    <p><b>アウストリヌス2世（修復王）</b>は内戦後の国家再建を主導。軍部の粛清と治安回復を実施し、以降の歴代王は軍事拡張よりも外交と安定を重視する政策へと舵を切った。</p>

    <h3 id="history-9">王権と議会の均衡：ユナニティス朝（14世紀 - 15世紀）</h3>
    <p>ルーマ8世による海軍強化と通貨改革により、経済的基盤が強化された。ギリス1世の時代には評議会の構成が拡大され、庶民の政治参加が本格化した。</p>

    <h3 id="history-10">民主化の進展と王朝の終焉（15世紀 - 16世紀）</h3>
    <p><b>ジースムンテア1世</b>により選挙制度が導入され、民主主義の重要な転換点を迎えた。しかし、16世紀後半には王権が再び衰退。エーレン6世の精神錯乱やギリス2世の戦死を経て、王朝は断絶の憂き目に遭った。</p>

    <h3 id="history-11">帝国編入期（1590年 - 1741年）</h3>
    <p>王政崩壊後、ユノヴァは<b>連合帝国</b>に編入された。約150年にわたる支配下で、ユノヴァ民族は政治的・文化的に抑圧され、言語や伝統の使用が厳しく制限された。この時期の抑圧が、後の強力な民族意識の源泉となった。</p>

    <h3 id="history-12">独立戦争と共和国の成立（1741年 - 1743年）</h3>
    <p>1741年、帝国の支配に対しユノヴァ民族が武装蜂起（<b>ユノヴァ独立戦争</b>）。1743年に独立を達成した。独立後、<b>アルヴェルト・イェルン</b>の指導のもとで現在の共和国が樹立された。イェルンは過去の王政と内戦の歴史を分析し、秩序と効率を重視した「ユノヴァ主義」に基づく国家設計を行った。</p>

    <h2 id="leader">初代大統領</h2>
    <h3>アルヴェルト・イェルン</h3>
    <p>ユノヴァ共和国の建国者であり、ユノヴァ主義の提唱者。特定の政党に寄らない「超越的指導者」として国家の精密な制度を設計した。彼の合理主義的な統治理念は、現在の共和国の隅々にまで浸透している。</p>
    <blockquote>
        「国家は人間のために存在するのではない。人間が国家のために存在する」
        <br>― アルヴェルト・イェルン
    </blockquote>
